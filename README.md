# 📝 CraftBoard – Notes Application

CraftBoard is a full-stack Notes Application built using the MERN stack that allows users to create, update, and delete notes with a smooth and responsive user experience.

🚀 Live Demo: https://craftboard-xii9.onrender.com
---

## 🚀 Features

* 🔐 User Authentication (JWT-based)
  * Register
  * Login
  * Logout
    
* 📝 Notes Management
  * Create Notes
  * Update Notes
  * Delete Notes
 
* 🔔 Real-time Notifications using 'react-hot-toast'

  * Note created successfully
  * Note updated successfully
  * Note deleted successfully
 

* ⚡ Rate Limiting for API protection

  
* 📱 Fully Responsive (Mobile + Desktop)

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* React Hot Toast

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

**Authentication:**

* JSON Web Tokens (JWT)

**Other:**

* Rate Limiting Middleware
